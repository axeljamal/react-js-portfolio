  ## Installation
  To reuse this template, you can clone the this repository:
  
          git@github.com:axeljamal/react-js-portfolio.git
          
          
  
  Make sure to install all the necessary dependency:
      
          npm intall  --
          
          
  To start the app:
  
          npm start
